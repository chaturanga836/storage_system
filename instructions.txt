# Tenant Node Instructions

## Protocol Buffer Generation
generate proto: python -m grpc_tools.protoc -I./proto --python_out=./tenant_node --grpc_python_out=./tenant_node ./proto/storage.proto

## Running the Tenant Node

### REST API Mode (Default)
python run.py
# or
python run.py rest

### gRPC Mode  
python run.py grpc

### Both REST and gRPC
python run.py both

## Demo and Testing
python demo.py

## Alternative Entry Points
python main.py

## Setup (Windows)
setup.bat

## Setup (Linux/Mac)
./setup.sh

## Configuration
Copy config.env.example to config.env and modify as needed.

## API Endpoints (REST)
- Health: GET http://localhost:8000/health
- Sources: GET http://localhost:8000/sources  
- Tenant Stats: GET http://localhost:8000/tenant/stats
- Write Data: POST http://localhost:8000/data/write
- Search Data: POST http://localhost:8000/data/search
- Aggregate: POST http://localhost:8000/data/aggregate
